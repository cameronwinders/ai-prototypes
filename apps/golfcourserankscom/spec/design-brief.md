# Design Brief

Target app: `apps/golfcourserankscom`

## Visual Direction

- Premium-trustworthy plus outdoorsy calm
- Editorial hierarchy over generic SaaS cards
- Restrained neutrals with one fairway-green accent
- Serif display treatment paired with a modern sans UI layer

## Interaction Priorities

- Mobile-first shell with one-handed bottom navigation
- Large tap targets and explicit drag handles
- Keyboard reorder controls with live announcements
- Inline save state, rollback messaging, and last-saved feedback
- Skeleton loading instead of blank screens

## Important Surfaces

- Home hero with leaderboard preview and quick search
- Filtered leaderboard with trust badges and participation counts
- Seeded course browse with clear played state
- Course detail with AI summary, disclaimer, metadata, and note editor
- My Courses split into ranked and needs-ranking views
- Friends page for invite, accept, and compare
- Feedback entry point in both nav and floating CTA
